Project Details

This project is built for classifying Intel images into various categories like buildings, sea, trees, etc. Here are the steps involved in building this project:

  • 1. Data Collection: A dataset of labeled images was collected for different classes (e.g., building, sea, tree).
  • 2. Preprocessing: The images were resized to 150x150 pixels and normalized for better model performance.
  • 3. Model Architecture: A Convolutional Neural Network (CNN) was built with layers like Conv2D, BatchNormalization, MaxPooling, and Dense layers for classification.
  • 4. Model Training: The model was trained using the categorical cross-entropy loss function and Adam optimizer for 70 epochs with batch size 128.
  • 5. Prediction: After training, the model was used to classify new images uploaded by users.
  • 6. Deployment: The application was deployed using Flask, allowing users to upload an image and get predictions through a web interface.
